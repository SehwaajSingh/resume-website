GitHub CLI install target:
https://cli.github.com/

If Windows package managers are unavailable, download the Windows 64-bit installer from:
https://github.com/cli/cli/releases/latest

After install, open a new terminal and run:
gh auth login

This repo is already prepared with:
- GitHub Pages workflow: .github/workflows/deploy-pages.yml
- Static Next.js export: next.config.ts
- Publish helper: publish-to-github.bat
